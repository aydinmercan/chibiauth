# ChibiAuth

Naive OpenID Provider.

## Development

We prefer commit signing via SSH. The allowed signers are listed in `.gitsigners`.
To verify, a gitconfig entry such as the following is likely needed:

```ini
[gpg.ssh]
    allowedSignersFile = .gitsigners
```
